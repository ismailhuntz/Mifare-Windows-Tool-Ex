# Lire un badge

Poser un badge sur le lecteur et cliquez sur  

![image](https://user-images.githubusercontent.com/3501675/73281824-d2094200-41f0-11ea-9918-91f196709a8c.png)

Ensuite, sélectionnez un fichier de clés de "mapping" (typiquement std.keys pour un badge vierge, ou bien un autre fichier de clés pour une carte spécifique)

![image](https://user-images.githubusercontent.com/3501675/73282630-35e03a80-41f2-11ea-9dbe-8c647b9d9652.png)

Ensuite, cliquez sur le bouton "Démarrer le mapping et lire le badge"

![image](https://user-images.githubusercontent.com/3501675/73282804-763fb880-41f2-11ea-8c48-d1a101283f93.png)

Si le badge est bien décodé, son dump sera automatiquement enregistré dans le répertoire dumps de l'application, avec l'UID du badge dans le nom du fichier. Par ex: mfc_83FA9871.dump
