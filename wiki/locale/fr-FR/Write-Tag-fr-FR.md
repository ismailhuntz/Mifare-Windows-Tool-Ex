## Dupliquer un badge

Vous pouvez écrire sur le bloc 0 (seulement pour les badges déverrouillés, dits "chinois").Pour cela, cliquez sur "Avancé : autoriser l'écriture sur le bloc 0"

Pour dupliquer un badge, sélectionnez d'abord un dump avec le bouton "Select Dump", ensuite choisissez un fichier de clés pour le badge cible (si le dump n'existe pas encore), puis attendez que le mapping soit créé, ensuite cliquez sur "Démarrer la copie"

L'option "Utiliser ces conditions d'accès pour tous les secteurs : FF0780" n'est pas encore disponible dans cet outil (nfc-mfcclassic, l'outil interne utilisé pour écrire le badge n'offre pas cette possibilité)

Vous pouvez ecrire un badge en utilisant la clé A ou bien la clé B, et tolérer les erreurs d'écriture ou bien l'interrompre en cas d'erreur.
